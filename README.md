# Landing Page: Local turístico

Projeto de **landing page de turismo** desenvolvido com **HTML e CSS** para Rocketseat como desafio prático dos **Fundamentos do HTML e do CSS**. A página apresenta informações sobre os principais pontos turísticos, imagens e dicas de viagem.

---

## Objetivo

- Praticar **HTML semântico** (header, section, footer)  
- Aplicar **CSS** para cores, fontes, bordas arredondadas e espaçamento  
- Apresentar informações de forma clara e visualmente agradável  

---

## Tecnologias

- HTML5  
- CSS3  
- Figma
